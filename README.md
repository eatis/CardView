アニメーションを扱うクラスはいくつかある。

その中で基本となるのがCABasicAnimation。

- プロパティを指定し、CABasicAnimationクラスのオブジェクト作成
- 初期値を指定
- 終了時の値を指定
- 時間を指定
- アニメーションをlayerに追加
